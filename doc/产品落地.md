# 你需要定义“什么值得报道”

例如：

## 值得报道：

### 1. AI能力变化

* 更强推理
* 更长上下文
* 更低价格
* 更强 Agent

---

### 2. 对行业有影响

例如：

* 替代某类岗位
* 降低创业成本
* 改变开发方式

---

### 3. 社区爆火

例如：

* GitHub Star暴涨
* Reddit疯狂讨论
* Twitter传播

---

# 不值得报道：

* “某AI公司融资”
* “某模型 benchmark 提升 2%”
* “营销软文”

用户根本不关心。
